# APP INTERNA MONOGNOMO 

## Descripción

Aplicación interna de la empresa Mono Gnomo para el registro de las horas dedicadas a cada uno de los proyectos realizados, la visualización de todos los proyectos realizados por meses, la gestión de los proyectos y la división del trabajo. Todo ello generando archivos CSV exportables y archivos PDF exportables para llevar el control de todos los aspectos empresariales relativos a los proyectos.
